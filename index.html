Tecnologias

Essas são as tecnologias que eu uso no meu dia de trabalho. Me sinto avontade em usalas e posso dizer que os meus conhecimentos nessas tecnologias são de intermediário / avançado

HTML

"Linguagem de Marcação de Hipertexto" é uma linguagem de marcação utilizada na construção de páginas na Web.

CSS

Folha de Estilo em Cascatas. É usado para estilizar elementos escritos em uma linguagem de marcação como HTML.

JavaScript

JS é uma linguagem de programação de uso geral, aplicada principalmente para desenvolvimento web e desenvolvimento de software.

Bootstrap

Bootstrap é um framework para desenvolvimento front-end para sites e aplicações web usando HTML, CSS e JavaScript
